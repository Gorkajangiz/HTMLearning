Iniciar Sesión

Ejercicio 1.1:

Para este primer ejercicio, vamos a copiar el formato inicial de creación de cualquier página web, recuerda que el código es el siguiente: 


        <!DOCTYPE html>
        <html lang="es-ES">
        <head>
          <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
          </head>
          <body>
            <header>
                
            </header>
            <main>
                  
            </main>
            <footer>
                          
            </footer>
          </body>
        </html>


Ejercicio 1.2:

Una vez hecho eso, prueba a añadir etiquetas como <h1>,<h2>,<h3>,<p> y <pre>, además de eso, siéntete libre de probar todas las etiquetas que quieras, como <img>,<a> o <span>:



        <!DOCTYPE html>
        <html lang="es-ES">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
          </head>
          <body>
            <header></header>
            <main>
              <h1>Estoy escribiendo un titulo</h1>
              <h3>Y ahora un subtitulo</h3>
              <p>Aquí añadiré mi información</p>
              <pre>Y aquí añadiré todavía más</pre>
             </main>
             <footer></footer>
          </body>
        </html>
