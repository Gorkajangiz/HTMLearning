Guía de Etiquetas HTML

  • <html>

    Define el elemento raíz de un documento HTML.

  • <head>

    Contiene metadatos sobre el documento HTML.

  • <title>

    Especifica el título del documento, que aparece en la pestaña del navegador.

  • <base>

    Define la URL base para todos los enlaces relativos del documento.

  • <meta>

    Proporciona metadatos como descripción, codificación de caracteres, palabras clave, etc.

  • <link>

    Conecta el documento con recursos externos, como hojas de estilo.

  • <style>

    Permite definir estilos CSS directamente en el documento.

  • <script>

    Permite incluir o enlazar scripts de JavaScript.

  • <body>

    Contiene el contenido visible del documento HTML.

  • <header>

    Define el encabezado de una página o sección.

  • <footer>

    Define el pie de página de una página o sección.

  • <main>

    Representa el contenido principal de una página.

  • <section>

    Agrupa contenido relacionado en una sección temática.

  • <article>

    Representa contenido independiente y autocontenido.

  • <aside>

    Define contenido relacionado con el principal, como una barra lateral.

  • <nav>

    Define un conjunto de enlaces de navegación.

  • <div>

    Contenedor genérico para agrupar elementos con fines de diseño o funcionalidad.

  • <p>

    Representa un párrafo de texto.

  • <h1> a <h6>

    Define encabezados, desde el más importante (<h1>) hasta el menos (<h6>).

  • <span>

    Contenedor en línea para aplicar estilos o clases.

  • <strong>

    Define texto importante con énfasis fuerte (normalmente se muestra en negrita).

  • <em>

    Define texto con énfasis (normalmente se muestra en cursiva).

  • <br>

    Inserta un salto de línea.

  • <hr>

    Inserta una línea horizontal para separar contenido.

  • <a>

    Crea un hipervínculo a otra página o recurso.

  • <img>

    Incrusta una imagen en el documento.

  • <table>

    Define una tabla de datos.

  • <form>

    Define un formulario interactivo.

  • <ul>

    Define una lista desordenada.

  • <ol>

    Define una lista ordenada.

  • <li>

    Define un elemento de una lista.

  • <dl>

    Define una lista de definiciones.

  • <dt>

    Define un término dentro de una lista de definiciones.

  • <dd>

    Define la descripción de un término en una lista de definiciones.

  • <blockquote>

    Define una cita larga.

  • <q>

    Define una cita corta en línea.

  • <cite>

    Representa el título de una obra creativa.

  • <abbr>

    Define una abreviatura o acrónimo.

  • <address>

    Define información de contacto.

  • <b>

    Aplica un estilo de negrita sin implicar importancia.

  • <i>

    Aplica un estilo de cursiva sin implicar énfasis.

  • <small>

    Reduce el tamaño del texto.

  • <mark>

    Resalta texto como marcado.

  • <del>

    Indica texto eliminado.

  • <ins>

    Indica texto insertado.

  • <sub>

    Representa texto como subíndice.

  • <sup>

    Representa texto como superíndice.

  • <code>

    Representa un fragmento de código.

  • <pre>

    Representa texto preformateado.

  • <var>

    Define una variable en un contexto matemático o de programación.

  • <samp>

    Representa una salida de computadora.

  • <kbd>

    Representa una entrada del teclado.

  • <time>

    Representa una fecha u hora específica.

  • <progress>

    Representa el progreso de una tarea.

  • <meter>

    Representa un valor en un rango conocido.

  • <fieldset>

    Agrupa elementos relacionados dentro de un formulario.

  • <legend>

    Proporciona un título a un elemento <fieldset>.

  • <label>

    Asocia un texto con un control de formulario.

  • <input>

    Define un control interactivo para formularios.

  • <button>

    Define un botón clicable.

  • <select>

    Define un menú desplegable.

  • <option>

    Define una opción dentro de un menú desplegable.

  • <textarea>

    Define un campo de texto multilínea.

  • <optgroup>

    Agrupa opciones relacionadas dentro de un menú desplegable.

  • <output>

    Representa el resultado de un cálculo.

  • <canvas>

    Proporciona un área para gráficos dinámicos.

  • <svg>

    Permite dibujar gráficos vectoriales.

  • <iframe>

    Incrusta otro documento HTML dentro de la página.

  • <details>

    Define un elemento de contenido expandible.

  • <summary>

    Proporciona un resumen para el elemento <details>.

  • <dialog>

    Representa un cuadro de diálogo interactivo.

  • <audio>

    Incrusta contenido de audio.

  • <video>

    Incrusta contenido de video.

  • <source>

    Especifica recursos multimedia alternativos.

  • <track>

    Proporciona subtítulos o pistas de texto para <video> o <audio>.

  • <map>

    Define un mapa de imagen con áreas clicables.

  • <area>

    Define una región dentro de un mapa de imagen.

  • <noscript>

    Proporciona contenido alternativo si JavaScript está deshabilitado.

  • <embed>

    Incrusta un contenido externo (como un complemento).

  • <object>

    Incrusta un recurso externo como una imagen o documento.

  • <param>

    Define parámetros para un objeto incrustado.

  • <marquee>

    Crea texto o contenido que se desplaza automáticamente, ya sea horizontal o verticalmente. Es una etiqueta obsoleta y no se recomienda su uso en HTML5.
