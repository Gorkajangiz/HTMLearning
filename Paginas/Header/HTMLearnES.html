Algunos proyectos de nuestros alumnos

Bricklink

Este alumno, apasionado coleccionista de Lego, decidió utilizar la idea de una ya existente página web para rediseñar por completo su estilo y darles un toque innovador. Tras muchas horas de trabajo y varias supervisiones por nuestra parte, logró crear lo que ves, una página completamente funcional llena de coloridos juguetes y vinculos a otras páginas que crean un ecosistema perfecto. Tiempo después de finalizar el proyecto, cerró un trato con la empresa a la que pertenecía la página de Bricklink original y logró vendersela, más tarde sustituyendo el anterior diseño por este nuevo y agradable entorno.

¿Quieres aprender pero no sabes por donde empezar?

Estos son los tres lenguajes principales para crear páginas web:

HTML:

El lenguaje universal sobre el que esta escrito el código principal de la mayoría de sitios web, y el mejor punto por el que empezar a aprender.

Información Atributos Aprender

CSS:

Una vez aprendido el código necesario para crear una web, toca hacerla lo suficientemente bonita para que la quieran visitar.

Información Atributos Aprender

JavaScript:

¿Quieres añadir funcionalidades que HTML y CSS no soportan? JavaScript es el lenguaje para darle a tu proyecto un extra de profesionalidad.

No hay de que avergonzarse, ¡Todos empezamos por algun lado! Ven con nosotros a la sección de ayuda y tutoría especializada para asegurarnos de que recibes la mejor educación posible

Nivel principiante:

No tienes ninguna experiencia con lenguajes de programación, no tienes conocimientos previos de HTML ni CSS.

¿Crees tener experiencia con HTML y CSS? Ahora que ya sabes las bases podemos saltarnos los tutoriales básicos y saltar directamente al conocimiento avanzado que necesitas para profesionalizar tus páginas

Nivel Intermedio:

Conoces HTML y CSS, tienes algún tipo de conocimiento al respecto y vienes a mejorar y perfeccionar tus bases.

¡Nivel avanzado! Debes ser muy bueno, pero siempre hay hueco para mejorar, y las páginas profesionales las hacen programadores profesionales

Nivel Avanzado:

Sabes manejar HTML y CSS, puedes crear y gestionar páginas y vienes a conseguir los ultimos pasos de aprendizaje como programador de páginas web.
