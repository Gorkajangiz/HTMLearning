Ejercicio 1.1:

Para este primer ejercicio, vamos a copiar los estilos más básicos de cualquier página web: 


         body {
         font-family: Arial, sans-serif;
         background-color: #f4f4f4;
         color: #333;
                  }
                  h1 {color: #3498db;text-align: center;}
                  h3 {color: #2ecc71;margin-left: 20px;} /*¡El texto tambien puede ponerse así!*/
                  p {
                    font-size: 16px;
                    line-height: 1.5;
                  }
                  pre {
                    background-color: #333;
                    color: #fff;
                    padding: 15px;
                    border-radius: 5px;
                  }


Ejercicio 1.2:

Una vez hecho eso, prueba a añadir etiquetas más complejas o a modificar elementos que no requieran nombrar sus clases, como <main>,<body>,<section> y <footer>, además de eso, siéntete libre de probar todas las etiquetas que quieras:


        body {
          font-family: 'Roboto', sans-serif;
          background-color: #eaeaea;
          color: #333;
          margin: 0;
          padding: 0;
        }
        header {
          background-color: #3498db;
          padding: 20px;
          text-align: center;
          color: white;
        }
        header h1 {
          font-size: 2.5em;
          margin: 0;
        }
        main {
          padding: 20px;
          display: flex;
          justify-content: space-around;
          flex-wrap: wrap;
        }
        section {
          background-color: white;
          width: 45%;
          margin: 10px;
          padding: 20px;
          border-radius: 8px;
          box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
        }
        section h2 {
          color: #3498db;
        }
        section p {
          line-height: 1.6;
        }
        footer {
          background-color: #2c3e50;
          color: white;
          text-align: center;
          padding: 10px 0;
          position: fixed;
          width: 100%;
          bottom: 0;
        }
        footer a {
          color: #ecf0f1;
          text-decoration: none;
          margin: 0 10px;
        }
        footer a:hover {
          text-decoration: underline;
        }
